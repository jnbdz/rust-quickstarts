# Operators

## Arithmetic
- `+`
- `-`
- `*`
- `/`
- `%`

## Relational
- `>`
- `<`
- `>=`
- `<=`
- `==`
- `!=`

Logical: 
- `&&`
- `||`
- `!`

## Bitwise
- `&` (AND)
- `|` (OR)
- `^` (XOR)
- `!` (not)
- `<<` (left shift)
- `>>` (right shift)
- `>>>` (right shift with zero)
